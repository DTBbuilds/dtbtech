API Service Information

This is a frontend-only deployment. API endpoints are handled gracefully with local storage fallbacks and proper error handling.

🔧 Technical Details

  • Mode: Static Frontend Deployment
  • Analytics: Local storage with sync capability
  • API Calls: Gracefully handled with fallbacks
  • Performance: Optimized for speed and reliability

📊 Available Services

  • Web Vitals monitoring (local storage)
  • Performance metrics collection
  • User interaction tracking
  • Error logging and reporting

🚀 For Developers

To enable full API functionality:

  • Set VITE_ANALYTICS_ENABLED=true
  • Configure VITE_API_URL with your backend
  • Deploy backend services to handle API endpoints
